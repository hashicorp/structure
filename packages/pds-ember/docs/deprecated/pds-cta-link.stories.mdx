# CtaLink [⚠️ deprecated ⚠️]

<img alt="Old Pds::CtaLink vs new Hds::Button or Hds::Link::Standalone or Hds::Link::Inline" src={ImageFile} width="900" height="235" />

The component `Pds::CtaLink` is now deprecated, and not available anymore. Please use one of the equivalent `Hds::Button`, `Hds::Link::Standalone`, or `Hds::Link::Inline` component:
- ["Button" in Figma](https://www.figma.com/file/noyY6dUMDYjmySpHcMjhkN/HDS-Product---Components?node-id=2340%3A21001&)
- [&lt;Hds::Button&gt; in Code](https://helios.hashicorp.design/components/button)
- ["Link standalone" in Figma](https://www.figma.com/file/noyY6dUMDYjmySpHcMjhkN/HDS-Product---Components?node-id=2365%3A21590)
- [&lt;Hds::Link::Standalone&gt; in Code](https://helios.hashicorp.design/components/link/standalone)
- ["Link inline" in Figma](https://www.figma.com/file/noyY6dUMDYjmySpHcMjhkN/HDS-Product---Components?node-id=2365%3A21590)
- [&lt;Hds::Link::Inline&gt; in Code](https://helios.hashicorp.design/components/link/inline)

For more information speak with the [HashiCorp Design System team](https://hashicorp.slack.com/archives/C7KTUHNUS).
