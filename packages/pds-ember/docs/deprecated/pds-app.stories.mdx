# App [⚠️ deprecated ⚠️]

The following "app" component and sub-components are now entirely deprecated, and not available anymore:
- `Pds::App`
- `Pds::App::Banner`
- `Pds::App::Body`
- `Pds::App::Drawer`
- `Pds::App::Footer`
- `Pds::App::Header`
- `Pds::App::ModalArea`
- `Pds::App::Sidebar`

They've been replaced with the new [hc-app-frame](https://github.com/hashicorp/cloud-ui/tree/main/addons/hc-nav/app/components/hc-app-frame) and [hc-nav](https://github.com/hashicorp/cloud-ui/tree/main/addons/hc-nav) in Cloud UI.

For more information speak with the maintainers of Cloud UI.
