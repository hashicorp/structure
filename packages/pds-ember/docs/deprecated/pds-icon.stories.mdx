# Icon [⚠️ deprecated ⚠️]

<img alt="Old Pds::Icon vs new FlightIcon" src={ImageFile} width="900" />

The component `Pds::Icon` (and the corresponding [Structure icons](https://github.com/hashicorp/structure-icons/)) is now deprecated, and not available anymore. Please use the equivalent `FlightIcon` component:
- [Flight icons website](https://flight-hashicorp.vercel.app/)

For more information speak with the [HashiCorp Design System team](https://hashicorp.slack.com/archives/C7KTUHNUS).
