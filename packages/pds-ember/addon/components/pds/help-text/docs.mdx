# Help Text

A visual style for blocks of text that provide additional context to the user.

- [Usage](#usage)
- [Styling](#styling)
- [Markup](#markup)
- [Accessibility](#accessibility)
- [See Also](#see-also)


## Usage
```hbs
<Pds::HelpText
 @variant={{string ['']}}
>
  <!-- text content -->
</Pds::HelpText>
```


## Styling

```scss
@use "pds/components/help-text";
```


## Markup
The `<Pds::HelpText>` Ember component handles generation of semantic HTML markup.

### Modifier Classes
For bold help text.

* @variant
  * **bold**: `pds-helpText--bold`
