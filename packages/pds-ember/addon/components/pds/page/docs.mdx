## Usage

```handlebars
<Pds::Page as |P|>
  <P.Header>...</P.Header>
  <P.Body>...</P.Body>
</Pds::Page>
```
* `<P.Header>` is an alias for `<Pds::PageHeader>`
* `<P.Body>` is a wrapper for body content.
