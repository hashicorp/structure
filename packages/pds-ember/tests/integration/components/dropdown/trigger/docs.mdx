# Dropdown::Trigger

`Pds::Dropdown::Trigger` renders a `<summary>` for use within a `Pds::Dropdown`
component.

## Accessibility
- `<Pds::Dropdown::Trigger>` renders a `<summary>` element, which acts similar
  to a `<button>` but _cannot_ be semantically `:disabled`.
