# CheckboxField
The `<Pds::CheckboxField>` component generates semantic markup for defining a
single checkbox field (i.e., input + label).

- [Usage](#usage)
- [Styling](#styling)
- [Markup](#markup)
- [Accessibility](#accessibility)
- [See Also](#see-also)


## Usage

```handlebars
<Pds::CheckboxField
  @invalid={{boolean}}
  @id={{string}}
>
  checkbox label
</Pds::CheckboxField>
```
