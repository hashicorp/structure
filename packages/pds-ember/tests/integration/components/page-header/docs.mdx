## Usage

```handlebars
<Pds::PageHeader as |H|>
  <H.Breadcrumbs>...</H.Breadcrumbs>
  <H.Title>...</H.Title>
  <H.Actions>...</H.Actions>
  <H.Tabs>...</H.Tabs>
</Pds::PageHeader>
```
* All subcomponents are wrappers for their respective content.
