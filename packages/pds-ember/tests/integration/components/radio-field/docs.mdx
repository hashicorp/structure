# RadioField
The `<Pds::RadioField>` component generates semantic markup for defining a
single radio field (i.e., input + label).

- [Usage](#usage)
- [Styling](#styling)
- [Markup](#markup)
- [Accessibility](#accessibility)
- [See Also](#see-also)


## Usage

```handlebars
<Pds::RadioField
  @invalid={{boolean}}
  @id={{string}}
>
  radio label
</Pds::RadioField>
```
