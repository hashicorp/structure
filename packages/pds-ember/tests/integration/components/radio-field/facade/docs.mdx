# RadioField::Facade
The `<Pds::RadioField::Facade>` subcomponent is a non-interactive, visual
facade designed to be used within the `<Pds::RadioField>` component, in
order to apply contextual styling that reflects the logical state of an HTML
radio control (i.e., `<input type="radio">`).

## Usage

```handlebars
<Pds::RadioField::Facade />
```
<ArgsTable of="PdsRadioFieldFacade" />


## Styling
```scss
@use "pds/components/radio-field/facade";
```

CSS styles are intended to be applied in the context of `<Pds::RadioField>`.

* **NOTE:** A radio `<input>` may still match `:disabled:active` if an
  associated `<label>` is being pressed.

### Theming

* **facade**
  * `--pds-radioFacade-backgroundColor`
  * `--pds-radioFacade-borderColor`
  * `--pds-radioFacade-boxShadow`
  * `--pds-radioFacade-color`
  * `--pds-radioFacade-cursor`
* **checkmark** (i.e., "tick")
  * `--pds-radioFacade-tick-visibility`
