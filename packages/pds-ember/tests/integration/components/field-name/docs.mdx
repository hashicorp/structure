# Field Name

A visual style to communicate the name of a field or fieldset.

- [Usage](#usage)
- [Styling](#styling)
- [Markup](#markup)

## Usage
No Ember assets are available.

Continue reading to learn how to apply semantic markup and styling.


## Styling
```scss
@use "pds/components/field-name";
```


## Markup

1. Apply the `.pds-fieldName` class to a semantic `<label>` or `<legend>`.

```html
<label class="pds-fieldName">...</label>

<legend class="pds-fieldName">...</legend>
```
